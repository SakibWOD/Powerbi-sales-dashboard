# Power BI Sales Dashboard

## Project Overview
This project is an interactive Power BI Sales Dashboard developed to analyze sales performance, customer behavior, and business trends using data visualization techniques.

## Dashboard Features
- Sales KPI Cards
- Regional Sales Analysis
- Category-wise Sales Distribution
- Monthly Sales Trend
- Customer Insights
- Top Products Analysis
- Interactive Filters & Navigation
- Dark Theme Professional UI

## Tools & Technologies
- Power BI
- SQL
- Excel

## Key Business Insights
- Dhaka generated the highest revenue among all regions.
- Electronics contributed the largest share of total sales.
- Sales peaked in January and gradually declined in later months.
- Top customers significantly impacted overall revenue growth.
